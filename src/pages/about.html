Как запустить проект локально?

  1. Сделать копию репозитория: git clone https://github.com/sarkisovTu/web-frontend-itmo.git
  2. Установить необходимые завиимости: npm install
  3. Запустить сборку проекта с помощью Webpack: npm run build
  4. После заверешния сборки перейти в директорию бандла: cd build
